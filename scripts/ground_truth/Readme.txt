		=== 10k_ground-truth.sh ===

This is a simple script which can be used to remove noise from
the measurements in the 10k and 100k datasets in order to make
a ground truth dataset and ground truth solution if processed.


		=== fix_cityTrees10k_ground-truth.sh ===

This is another simple script that fixes indexing in the city-
Trees10k dataset. You will need the noisy cityTrees10k.txt and
the ground truth file cityTrees10000_groundtruth.txt which can
be found in SLAM++ data files and in iSAM OpenSLAM repository,
respectively.
